# Debugging Outputs

<!-- Source: https://swatplus.gitbook.io/io-docs/swat+-output-files/debugging-outputs -->

These output files are printed for every simulation. They are used by the SWAT+ developers for for debugging the National Agroecosystems Model (NAM). Their usefulness for SWAT+ users is probably very limited, so there is no detailed description of the individual files included here. Their names are:

- simulation.out
- success.fin
- files\_out.out
- drareas.out
- diagnostics.out
- checker.out
