# pest_water.ini

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/initialization/pest_water.ini -->

| Field    | Description                                                                                           | Type   | Unit | Default | Range |
| -------- | ----------------------------------------------------------------------------------------------------- | ------ | ---- | ------- | ----- |
| name     | Name of initial soil plant                                                                            | string | n/a  | n/a     | n/a   |
| sw\_frac | Fraction of soil water                                                                                | real   | n/a  |         | 0-1   |
| nut      | Filename points to the name column in the nutrients.sol file. If not used in the model, enter 'null'. | string | n/a  | n/a     | n/a   |
| pest     | Filename points to the name column in the pest\_hru.ini file. If not used in the model, enter 'null'. | string | n/a  | n/a     | n/a   |
| path     | Filename points to the name column in the path\_hru.ini file. If not used in the model, enter 'null'. | string | n/a  | n/a     | n/a   |
| salt     | Filename points to the name column in the salt\_hru.ini file. If not used in the model, enter 'null'. | string | n/a  | n/a     | n/a   |
| hmet     | Filename points to the name column in the hmet\_hru.ini file. If not used in the model, enter 'null'. | string | n/a  | n/a     | n/a   |
