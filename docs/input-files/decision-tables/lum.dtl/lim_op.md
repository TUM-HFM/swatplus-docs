# lim_op

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/decision-tables/lum.dtl/lim_op -->

The limit operators available and the limit and condition variables they are used for are listed in the table below.

| Operator | Description                                 | Limit variable                                         |
| -------- | ------------------------------------------- | ------------------------------------------------------ |
| \*       | Multiply limit variable by limit constant   | fc, wp, ul (soil\_water) evol, pvol (vol and vol\_wet) |
| +        | Add limit constant to limit variable        | fc, wp, ul (soil\_water) evol, pvol (vol and vol\_wet) |
| -        | Subtract limit constant from limit variable | fc, wp, ul (soil\_water) evol, pvol (vol and vol\_wet) |
| /        | Divide limit variable by limit constant     | fc, wp, ul (soil\_water) evol, pvol (vol and vol\_wet) |
