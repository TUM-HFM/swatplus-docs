# obj_num

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/decision-tables/lum.dtl/obj_num-1 -->

If *obj\_num* = 0, the action will be performed for all objects of the type specified by [obj](obj.md). If *obj\_num* > 0, the action will be performed for the object with the specified ID only.
