# Wetlands

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/wetlands -->

Wetlands are HRUs in which ponding of water on the surface is allowed. There are two files that are specific to wetlands:

- Main wetland file: [**wetland.wet**](wetland.wet.md)
- Wetland hydrology: [**hydrology.wet**](hydrology.wet.md)

For defining the remaining wetland properties, the following reservoir files are used:

- Wetland initialization: [**initial.res**](../reservoirs/initial.res.md)
- Wetland sediment: [**sediment.res**](../reservoirs/sediment.res.md)
- Wetland nutrients: [**nutrients.res**](../reservoirs/nutrients.res.md)
- Wetland release: [**res\_rel.dtl**](../decision-tables/lum.dtl.md)
