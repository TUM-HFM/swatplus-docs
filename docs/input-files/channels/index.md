# Channels

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/channels -->

There are a number of SWAT+ files that control the simulation of channel processes. The file [**channel-lte.cha**](channel-lte.cha.md) summarizes the main channel information and references several other files that specify the details:

- [**initial.cha**](initial.cha.md) references several files, which specify the initial contents of nutrients and constituents in the water,
- [**hyd-sed-lte.cha**](hyd-sed-lte.cha.md) controls the channel hydrology and sediment properties, and
- [**nutrients.cha**](nutrients.cha.md) controls the channel nutrient properties.
