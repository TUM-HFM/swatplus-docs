# Hydrology

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/hydrology -->

There are four files containing the basic topographical and hydrological properties of HRUs:

- [topography.hyd](topography.hyd.md) defines the topographic characteristics of the HRUs and routing units,
- [hydrology.hyd](hydrology.hyd.md) defines the hydrological characteristics of the HRUs,
- [field.fld](field.fld.md) specifies the properties of representative fields,
- [snow.sno](snow.sno.md) controls the simulation of snowfall and snowmelt processes.
