# Hydrologic Response Units

If regular HRUs are used in a SWAT+ setup, there is one main file, [**hru-data.hru**](../hydrologic-response-units/hru-data.hru.md), which references several other files specifying the HRU properties. The properties of HRU-lte objects are defined in [**hru-lte.hru**](../hydrologic-response-units/hru-lte.hru.md).
