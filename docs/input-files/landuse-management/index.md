# Landuse and Management

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/landuse-and-management -->

There are several SWAT+ files that control the simulation of land use and management. The file [landuse.lum](../landuse-and-management/landuse.lum.md) summarizes the main land use information and references several other files that specify the details:

- [plant.ini](../landuse-and-management/plant.ini.md) stores information about the plants growing in a rotation or plant community,
- [management.sch](../landuse-and-management/management.sch.md) is used to schedule management operations by heat units or dates and/or to list the decision tables to use for scheduling and conditioning management operations,
- [lum.dtl](../decision-tables/lum.dtl.md) contains the land use and management decision tables,
- [cntable.lum](../landuse-and-management/cntable.lum.md) lists typical Curve Number values for different land use types,
- [cons\_practice.lum](../landuse-and-management/cons_practice.lum.md) lists the USLE P values and slope lengths for various conservation practices,
- [ovn\_table.lum](../landuse-and-management/ovn_table.lum.md) lists overland Manning's n values for different tillage and land cover types.
