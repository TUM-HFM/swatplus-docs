# name (sweep.ops)

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/management-practices/sweep.ops/name_sweep -->

The name of the street sweeping operation is a primary key referenced when street sweeping operations are scheduled in [**management.sch**](../../landuse-and-management/management.sch.md) or [**lum.dtl**](../../decision-tables/lum.dtl.md). All names in the **sweep.ops** file must be unique.

| Name      | Description              |
| --------- | ------------------------ |
| high\_eff | High-efficiency sweeping |
