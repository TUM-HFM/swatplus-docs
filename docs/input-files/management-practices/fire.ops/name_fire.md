# name (fire.ops)

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/management-practices/fire.ops/name_fire -->

The name of the fire operation is a primary key referenced when burning/fire operations are scheduled in [**management.sch**](../../landuse-and-management/management.sch.md) or [**lum.dtl**](../../decision-tables/lum.dtl.md). All names in the **fire.ops** file must be unique.

| Name          | Description         |
| ------------- | ------------------- |
| grass         | Grass fire          |
| tree\_intense | Intense forest fire |
| tree\_low     | Low forest fire     |
