# name (irr.ops)

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/management-practices/irr.ops/name_irr -->

The name of the irrigation operation is a primary key referenced when irrigation operations are scheduled in [**management.sch**](../../landuse-and-management/management.sch.md) or [**lum.dtl**](../../decision-tables/lum.dtl.md). All names in the **irr.ops** file must be unique.

| Name            | Description       |
| --------------- | ----------------- |
| sprinkler\_med  | Medium sprinkler  |
| sprinkler\_high | High sprinkler    |
| drip            | Drip irrigation   |
| furrow          | Furrow irrigation |
