# name (hru-data.hru)

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/hydrologic-response-units/hru-data.hru/name_hru -->

The SWAT+ Editor assigns a name to every object, which is typically the abbreviation of the object type followed by the object number in QSWAT+. However, the name of the HRU is not used by SWAT+, so the user may change it.
