# name (cons_practice.lum)

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/landuse-and-management/cons_practice.lum/name_consp -->

The name of the conservation practice is a primary key referenced by [cons\_prac](../landuse.lum/cons_prac.md) in [**landuse.lum**](../landuse.lum.md). All names in the **cons\_practice.lum** file must be unique.
