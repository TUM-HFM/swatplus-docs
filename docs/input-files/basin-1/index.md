# Basin

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/basin-1 -->

General watershed attributes are defined in two basin input files, [**codes.bsn**](codes.bsn.md) and [**parameters.bsn**](parameters.bsn.md). These attributes control a diversity of physical processes at the watershed level. Users can use the default values set by the interfaces or change them to better reflect what is happening in a given watershed.
