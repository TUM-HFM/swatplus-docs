# name (initial.res)

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/reservoirs/initial.res/name -->

The name of the reservoir and wetland initialization record is a primary key referenced by the foreign keys [init](../reservoir.res/init.md) in [**reservoir.res**](../reservoir.res.md) and [init](../../wetlands/wetland.wet/init.md) in [**wetland.wet**](../../wetlands/wetland.wet.md). All names in the **initial.res** file must be unique.
