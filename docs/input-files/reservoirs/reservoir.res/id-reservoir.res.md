# id (reservoir.res)

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/reservoirs/reservoir.res/id-reservoir.res -->

The ID of the reservoir is a primary key referenced by the foreign key [res](../../connectivity/hru.con/hru.md) in [**reservoir.con**](../../connectivity/hru.con.md). All IDs in the **reservoir.res** file must be unique.
