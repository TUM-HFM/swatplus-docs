# yr_op

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/reservoirs/hydrology.res/yr_op -->

The year specified here must be within the simulation period. If 0 is input for *yr\_op* and [mon\_op](mon_op.md), the model assumes the reservoir is in operation at the beginning of the simulation.
