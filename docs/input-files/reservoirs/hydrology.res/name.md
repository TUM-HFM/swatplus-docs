# name (hydrology.res)

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/reservoirs/hydrology.res/name -->

The name of the reservoir hydrology record is a primary key referenced by the foreign key [hyd](../reservoir.res/hyd.md) in [**reservoir.res**](../reservoir.res.md). All names in the **hydrology.res** file must be unique.
