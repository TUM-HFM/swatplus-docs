# name (nutrients.res)

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/reservoirs/nutrients.res/untitled -->

The name of the reservoir and wetland nutrient record is a primary key referenced by the foreign keys [nut](../reservoir.res/nut.md) in [**reservoir.res**](../reservoir.res.md) and [nut](../../wetlands/wetland.wet/nut.md) in [**wetland.wet**](../../wetlands/wetland.wet.md). All names in the **nutrient.res** file must be unique.
