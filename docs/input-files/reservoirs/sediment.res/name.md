# name (sediment.res)

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/reservoirs/sediment.res/name -->

The name of the reservoir and wetland sediment record is a primary key referenced by the foreign keys [sed](../reservoir.res/sed.md) in [**reservoir.res**](../reservoir.res.md) and [sed](../../wetlands/wetland.wet/sed.md) in [**wetland.wet**](../../wetlands/wetland.wet.md). All names in the **sediment.res** file must be unique.
