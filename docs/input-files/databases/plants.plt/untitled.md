# name (plants.plt)

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/databases/plants.plt/untitled -->

The name of the plant/landcover is a primary key referenced by [plt\_name](../../landuse-and-management/plant.ini/plnt_name.md) in [**plant.ini**](../../landuse-and-management/plant.ini.md). All names in the **plants.plt** database must be unique.

The names in the plant database are also used by QSWAT+ to link the grid codes in land use/land cover maps to SWAT+ land use types.
