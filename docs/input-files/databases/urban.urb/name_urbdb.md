# name (urban.urb)

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/databases/urban.urb/name_urbdb -->

The name of the urban land type is a primary key referenced by [urban](../../landuse-and-management/landuse.lum/urb_ro.md) in [**landuse.lum**](../../landuse-and-management/landuse.lum.md). All names in the **urban.urb** database must be unique.

The names in the urban database are also used by QSWAT+ to link the grid codes in land use/land cover maps to SWAT+ urban land types.

| Name | Description                   |
| ---- | ----------------------------- |
| urhd | Residential - high density    |
| urmd | Residential - medium density  |
| urml | Residential - med/low density |
| urld | Residential - low density     |
| ucom | Commercial                    |
| uidu | Industrial                    |
| utrn | Transportation                |
| uins | Institutional                 |
| urbn | Generic                       |
