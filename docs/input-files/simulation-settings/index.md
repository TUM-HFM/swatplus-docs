# Simulation Settings

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/simulation-settings -->

There are three files that control the settings for a simulation run:

- [**object.cnt**](object.cnt.md) specifies the land area and the total area (including ponds and reservoirs) of the watershed and the counts of all spatial objects in a simulation,
- [**time.sim**](time.sim.md) controls the simulation time period and time step, and
- [**print.prt**](print.prt.md) controls which output files will be printed for a simulation.

An additional, optional file, [**object.prt**](object.prt.md), allows the user to print selected output for individual spatial objects.
