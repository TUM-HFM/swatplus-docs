# obj_typ_no

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/simulation-settings/object.prt/obj_typ_no -->

Soil water contents of the soil layers can be printed for all HRUs in the same file by setting *obj\_typ\_no* = 0. For all other outputs, a separate object print record with a unique name has to be defined for each object.
