# hydcon

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/simulation-settings/print.prt/hydcon -->

This output file is used by the SWAT+ developers for debugging connectivity errors and infinite loops. Its usefulness for SWAT+ users is probably very limited. Accordingly, there is no description of this output file included in the [SWAT+ Output Files](../../../output-files/output-file-format.md) section.
