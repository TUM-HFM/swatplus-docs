# aa_int_cnt

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/simulation-settings/print.prt/aa_int_cnt -->

If *aa\_int\_cnt* = 0, the model will print average annual output for the entire period. If *aa\_int\_cnt* > 0, the end years of the print intervals have to be specified by the user by listing them in chronological order in the same line as *aa\_int\_cnt*.
