# amount

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/water-allocation/water_allocation.wro/amount -->

If [withdr](withdr.md) = ave\_day, the amount needs to be specified here. If using a Decision Table or a recall file, *amount* should be set to 0, because the amount will be controlled by the DT or recall file.
