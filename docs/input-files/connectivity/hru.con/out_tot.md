# out_tot

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/connectivity/hru.con/out_tot -->

The number of outgoing hydrographs determines, how many spatial objects receive flow from an object. The next four columns ([obj\_typ](obj_typ.md), [obj\_id](obj_id.md), ​[hyd\_typ](hyd_typ.md), [frac](frac.md)) are repeated for each outgoing hydrograph.
