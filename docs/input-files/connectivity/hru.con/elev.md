# elev

<!-- Source: https://swatplus.gitbook.io/io-docs/introduction-1/connectivity/hru.con/elev -->

The elevation of HRUs is used to adjust precipitation and temperature when lapse rates are defined in [**parameters.bsn**](../../basin-1/parameters.bsn.md) ([plaps](../../basin-1/parameters.bsn/s_max.md) and [tlaps](../../basin-1/parameters.bsn/n_fix.md)). The elevation of the remaining objects is not used by the model.

Even though the elevation of most objects is not used by the model, a value has to be specified in the area column to make sure the file gets read correctly.
